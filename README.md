🔥 Smoke Detection System Using Machine Learning
📌 Overview

This project implements a machine learning-based smoke detection system that predicts fire alarms using real-world sensor data. The system analyzes environmental readings from multiple sensors to determine whether a fire alarm should be triggered, enabling early detection and enhanced safety monitoring.

🎯 Project Objectives

Predict fire alarms based on sensor readings

Identify the most important features affecting fire detection

Compare the performance of different machine learning models

Improve model efficiency through feature selection and preprocessing

📊 Dataset

Type: Sensor-based dataset

Description: Contains environmental readings from multiple sensors

Target Variable: Fire Alarm (Binary Classification: 0 = No Fire, 1 = Fire)

⚙️ Features Implemented
🔹 Data Preprocessing

Handled missing values

Applied StandardScaler for feature scaling

Cleaned and prepared data for modeling

🔹 Exploratory Data Analysis (EDA)

Analyzed feature distributions

Identified correlations between variables

Visualized key patterns in the dataset

🔹 Feature Selection

Random Forest Feature Importance

Recursive Feature Elimination (RFE)

Reduced dimensionality and improved model efficiency

🔹 Machine Learning Models

Random Forest Classifier

Additional classifiers for comparison

🔹 Model Evaluation

Train/Test split for generalization

Metrics used: Accuracy, Precision, Recall, F1 Score

🛠️ Tech Stack

Programming Language: Python

Libraries: Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn
