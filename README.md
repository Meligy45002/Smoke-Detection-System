🔥 Smoke Detection System using Machine Learning
📌 Overview
This project focuses on building a machine learning model to predict fire alarms using real-world sensor data. The system analyzes environmental readings to determine whether a fire alarm should be triggered, supporting early detection and safety monitoring.

🎯 Project Objectives
Predict fire alarms based on sensor readings
Identify the most important features affecting fire detection
Compare different machine learning models
Improve model performance through feature selection and preprocessing
📊 Dataset
Type: Sensor-based dataset
Description: Contains environmental readings collected from multiple sensors
Target Variable: Fire Alarm (Binary Classification: 0 = No Fire, 1 = Fire)
⚙️ Features Implemented
🔹 Data Preprocessing
Handled missing values
Applied feature scaling (StandardScaler)
Cleaned and prepared data for modeling
🔹 Exploratory Data Analysis (EDA)
Analyzed feature distributions
Identified correlations between variables
Visualized key patterns in the dataset
🔹 Feature Selection
Random Forest Feature Importance
Recursive Feature Elimination (RFE)
Reduced dimensionality and improved model efficiency
🔹 Machine Learning Models
Random Forest
Additional classification models for comparison
🔹 Model Evaluation
Train/Test split to ensure generalization
Evaluated using:
Accuracy
Precision
Recall
F1 Score
